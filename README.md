# Electron MD 便签

使用 [Electron](https://www.electronjs.org/) 和 [Vditor](https://b3log.org/vditor/?utm_source=ld246.com) 制作的简易便签

![](https://cdn.jsdelivr.net/gh/Tuzilow/blog-image/img/showmdnote.png)

## 使用方法

yarn install 安装依赖
yarn start 测试应用
yarn build 打包

## 功能

- Markdown 编辑器
- 窗口置顶
- 缓存上次的内容
